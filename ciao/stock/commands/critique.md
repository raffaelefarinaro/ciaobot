# Critique: $ARGUMENTS

Review the target critically.

Check:
- What assumption could be false?
- What requirement is missing or vague?
- What failure mode would be expensive later?
- What simpler option exists?
- What evidence would change the recommendation?

Return the highest-impact issues first, then concrete next steps.
